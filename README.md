# Toy-Vue3.x

手写 vue3 核心源码，理解其原理 by myself

## 🙌 目的

读懂 Vue3 源码可能是 2022 年要进大厂的必经之路了。

但是直接阅读源码的难度非常大，因为除了核心逻辑以外框架本身还要处理很多 edge case(边缘情况) 、错误处理、热更新等一系列工程问题。

为了学习源码，提高自己对 vue3 的理解，能在开发中规避一些不必要的 bug 以及迅速给出项目的优化方案，因此创建这个 repo。
